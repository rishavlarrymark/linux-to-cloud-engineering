4. Difference between kill -15 and kill -9
--------------------------------
kill -15 (SIGTERM):
Gracefully stops a process and allows cleanup.

kill -9 (SIGKILL):
Forcefully terminates a process without cleanup.

5. One-line command meanings
--------------------------------
ps      : Displays running processes.
top     : Shows real-time CPU and memory usage.
htop    : Interactive process viewer.
nice    : Starts a process with a given priority.
renice  : Changes priority of a running process.
